| 场景参考图命名 | `scene_{场景名}_{版本}.png`，如 `scene_stage_ruin.png`（仅作美术参考） |
| **子场景地图命名** | `map_{场景名}.json`（Tiled 导出），如 `map_stage_ruin.json`、`map_teahouse.json` |
| UI 命名 | `ui_{元素名}.png`，如 `ui_dialog_corner.png` |
| 精灵图集 | 同类型打包单张 spritesheet，间距 0px |
| Tiled 地图 | `.json` 格式（大地图1份 + 每个子场景独立1份） |
| 尺寸 | 严格遵循上述像素尺寸 |
