## 二、故事设定

> 你从大城市归来，接手父亲留下的即将解散的传统戏班。在这个江南小镇上，你需要在有限的时间内做出最终抉择——是重振旗鼓，还是任其消散。

- **主角**：归乡的戏班继承人
- **舞台**：一座江南小镇
- **核心冲突**：传统 vs 现代、传承 vs 放弃、自我认知的挣扎
- **主题**：玩家会在在剧情发展中逐步认识自己、认识中华戏曲，最终通过选择认识自己的性格与价值观

---

## 三、2D 俯视开放世界

### 地图设计
- **视角**：俯视 2D，整个小镇一张大地图
- **移动**：WASD 控制角色在小镇上自由移动
- **交互**：F 键与 NPC / 物品交互
- **关键地点**（可进入的子场景）：

| 地点 | 场景描述 |
|------|----------|
| 🍵 **茶馆** | 镇上消息集散地，各路人物往来 |
| 🏛️ **祠堂** | 家族记忆与文化根基所在 |
| 🎭 **戏台** | 戏班的核心，排练与演出场所 |
| ⛵ **码头** | 远方来客、物资流通的枢纽 |

- **场景切换**：在地图上走到地点入口 → 进入专属场景画面（不同背景/氛围）

### NPC 在地图中的表现
- 普通 NPC 散落在小镇各处，各自有专属动作（码头钓鱼、街边下棋、河边洗衣等）
- 玩家可以观察他们的日常生活，也可以上前对话

## 四、NPC 双层 Agent 架构（核心亮点）

### 重要 NPC — AI Agent 驱动

每个重要 NPC 是一个独立的 AI Agent，工作流程：

```
┌─────────────────────────────────────────────────┐
│              共享顶层状态 (Global Context)         │
│  · 世界观设定（时代背景、小镇历史、戏班故事）        │
│  · 当前游戏阶段（不屑 → 兴趣 → 抉择）              │
│  · 关键事件记录（已发生的重大剧情节点）              │
└──────────────────┬──────────────────────────────┘
                   │ 注入
    ┌──────────────┼──────────────┐
    ▼              ▼              ▼
┌─────────┐  ┌─────────┐  ┌─────────┐
│ Agent A │  │ Agent B │  │ Agent C │
│ 老票友  │  │ 竞争对手 │  │ 文化馆  │
│ 个性：   │  │ 个性：   │  │ 个性：   │
│ 怀旧感性 │  │ 精明现实 │  │ 理想主义 │
└────┬────┘  └────┬────┘  └────┬────┘
     │            │            │
     ▼            ▼            ▼
  结合顶层状态 + 自身人格 → 生成独有对话/行为
```

**关键机制：**
- Agent 先读取"顶层共享状态"（背景 + 当前阶段），再结合自身个性生成回应
- 同一件事，不同 Agent 反应完全不同
- 阶段推进时，所有 Agent 的输出基调同步变化

### 普通 NPC — 固定对话库 + 随机

- 预设对话库，多套台词随机抽取
- 各有专属动画/动作，增强小镇生活感
- 对重要剧情的推进不产生关键影响

## 五、玩家交互系统

### 交互流程

```
玩家遇到 NPC
    │
    ├── 手动输入文字（自由对话）
    │        └── AI 实时生成 NPC 回应 → 渲染展示
    │
    └── AI 生成对话选项（3-4个）
             └── 玩家选择一个 → NPC 回应
```

### 对话数据管理
- **全量存储**：所有对话记录持久化
- **关键节点总结**：到达新阶段/触发重要事件时，AI 自动对之前的对话做摘要归档
- **人物关系追踪**：记录玩家与每个重要 NPC 的关系变化

## 六、三阶段情感推进系统

| | 阶段一：不屑 | 阶段二：了解 | 阶段三：抉择 |
|------|:---:|:---:|:---:|
| **玩家心态** | 被迫接手，心不在焉 | 逐渐了解戏班与人，产生兴趣 | 面临最终选择 |
| **触发条件** | 游戏开始 | 积累足够 NPC 互动 / 关键事件 | 时间截止或剧情节点 |
| **Agent 输出基调** | 冷淡、疏离、简短 | 热情、开放、深入 | 两极化：挽留 or 告别 |
| **美术色调** | 冷色 / 灰暗 | 渐暖 / 色彩逐渐丰富 | 暖色（接受）or 暗沉（拒绝） |
| **音乐氛围** | 疏离、低沉 | 逐渐丰富、温暖 | 激昂（接受）or 感伤（拒绝） |

## 七、多结局 + AI 总结评价

### 结局分支
- **接受戏班**：留下来继承父业，重振戏班
  - 根据与不同 NPC 的关系深度，细分不同结局版本
- **拒绝戏班**：选择离开，回到城市生活
  - 根据离开前是否做好妥善安排，细分不同结局版本

### AI 结局评价
- 回顾整个故事中的**重点事件**
- 总结玩家的**关键选择**及**人际关系走向**
- 生成一段个性化的"人生感悟"——关于你是什么样的人、你对传统文化的态度
- 既是游戏总结，也是对玩家自身的映照

## 八、文化元素融入

- 可在戏班场景中**插入戏曲表演片段**（过场/可触发事件）
- NPC 对话中自然渗透**戏曲术语、经典剧目、行当知识**
- 不同地点有不同的文化氛围（茶馆 → 评弹/说书，祠堂 → 家族戏/祭祀戏）
