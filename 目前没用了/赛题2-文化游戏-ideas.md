# 赛题二：文化表达游戏 Ideas

> **背景：** 腾讯黑客松，CodeBuddy Agent 辅助，3人团队
> **引擎：** Godot / Unity

---

## Idea 1：《织梦江南》— 传统文化焕新（非遗手工艺）

| 属性 | 评估 |
|------|------|
| **类型** | 轻度 Roguelike + 手工艺模拟 |
| **难度** | ★★★☆☆（中） |
| **CodeBuddy 适配度** | ★★★★☆ — Rougelike 随机生成逻辑和工序小游戏的代码模板 AI 能高效产出 |
| **3人团队可行性** | ★★★☆☆ — 需要手绘美术风格，对美术能力有一定要求 |
| **独特性** | ★★★★☆ — 非遗手工艺+游戏的结合点在市场上很少见 |

**玩法：**
- 你是一个江南织坊的学徒，通过染布、刺绣、织造等工序制作作品去参加"市集"或"贡品选拔"
- 每关是一个手工艺工序小游戏（配色搭配→图案拼接→纹理组合），完成质量决定作品等级
- 融入真实非遗元素（宋锦、苏绣、蓝印花布），每件作品完成后展示文化背景
- **亮点：** 游戏本身就是传播非遗的过程
- **开发要点：** 2D 拼接/配对玩法，素材可用手绘风格，文化内容需提前调研

## Idea 2：《乡愁快递》— 城乡文化

| 属性 | 评估 |
|------|------|
| **类型** | 经营模拟 + 剧情叙事 |
| **难度** | ★★☆☆☆（低） |
| **CodeBuddy 适配度** | ★★★★☆ — 订单系统、背包管理、剧情触发逻辑 AI 都能高效搭建 |
| **3人团队可行性** | ★★★★★ — 2D 俯视角开发难度低，内容和剧情可批量填充 |
| **独特性** | ★★★★☆ — "城乡联结"+快递题材接地气且有时代感 |

**玩法：**
- 你在乡村开了一家"特产快递站"，帮在外打拼的年轻人把家乡味道寄到城市
- 接收订单 → 在乡村地图寻找对应的特产（食材/手工艺品）→ 打包 → 发货
- 每寄出一件特产，触发一段收件人的温情小故事（在城市吃到家乡味道的感动）
- **亮点：** 展现乡村振兴、城乡联结，题材有温度
- **开发要点：** 2D 俯视角探索 + 点击操作，重点在叙事内容

## Idea 3：《鼓乐节拍》— 民族地域特色

| 属性 | 评估 |
|------|------|
| **类型** | 节奏音游 |
| **难度** | ★★★☆☆（中） |
| **CodeBuddy 适配度** | ★★★☆☆ — 音游判定逻辑 AI 可写，但音频素材处理和节奏谱面编排需要人工 |
| **3人团队可行性** | ★★★☆☆ — 音乐版权/录制和谱面设计是瓶颈，编程工作相对少 |
| **独特性** | ★★★★★ — 少数民族鼓乐音游在市场上几乎没有竞品 |

**玩法：**
- 以中国少数民族传统鼓乐为核心，玩家跟随民族音乐的节奏击鼓
- 下落式音游（类似《节奏大师》），但鼓点来自真实的民族音乐（苗族鼓舞、藏族锅庄、维吾尔十二木卡姆等）
- 每通关一个曲目解锁该民族的文化知识卡片
- **亮点：** 用音游形式让年轻人接触民族音乐
- **开发要点：** 音游逻辑成熟，重点是合法获取民族音乐素材和谱面编排

## Idea 4：《节气厨房》— 二十四节气饮食文化

| 属性 | 评估 |
|------|------|
| **类型** | 烹饪节奏模拟 |
| **难度** | ★★★☆☆（中） |
| **CodeBuddy 适配度** | ★★★★☆ — 烹饪流程状态机和节奏判定逻辑 AI 能高效生成 |
| **3人团队可行性** | ★★★★☆ — 2D 手绘风格轻量开发，1人美术+2人程序/内容 |
| **独特性** | ★★★★★ — 节气+烹饪+节奏的混搭非常新鲜，市场无对标 |

**玩法：**
- 根据二十四节气的变换，制作对应的传统美食（立春咬春饼、冬至包饺子、端午包粽子…）
- 核心是烹饪节奏游戏 — 切菜、揉面、调味都有节奏判定，完美操作做出"时令佳品"
- 每个节气有不同的食材组合和烹饪手法，完成后展示该节气的文化内涵
- **亮点：** 美食+节气+节奏的独特组合，容易出圈传播
- **开发要点：** 节奏游戏 + 食谱拼接，美术以手绘风格为主，重点做好节气文化卡片

## Idea 5：《墨游记》— 中国水墨画

| 属性 | 评估 |
|------|------|
| **类型** | 点击解谜 + 水墨画交互 |
| **难度** | ★★★☆☆（中） |
| **CodeBuddy 适配度** | ★★★★☆ — 点击交互逻辑和关卡状态机 AI 可高效产出，水墨渲染 Shader 需定制 |
| **3人团队可行性** | ★★★☆☆ — 对美术风格要求高（水墨质感决定游戏成败），但编程工作量可控 |
| **独特性** | ★★★★★ — "走入名画"的概念在游戏中很少被实现，视觉辨识度极高 |

**玩法：**
- 玩家扮演一个误入水墨画卷的墨水精灵，需要在不同朝代的名画中穿梭（《千里江山图》《富春山居图》等），修复被破坏的画面
- 玩家用毛笔工具在画面上"补墨"、连线、配对来推进。每一幅名画是一个关卡，有独特的美术风格和谜题
- 修复过程中了解画作的历史背景和画家故事
- **亮点：** 交互式赏画，让玩家"进入"名画
- **开发要点：** 2D 点击解谜，Shader 实现水墨晕染效果，文化内容需提前调研

## Idea 6：《戏台争霸》— 戏曲文化

| 属性 | 评估 |
|------|------|
| **类型** | 卡牌对战 |
| **难度** | ★★★★☆（中高） |
| **CodeBuddy 适配度** | ★★★★★ — 卡牌系统（抽牌、出牌、效果结算、AI对手）是 AI 最擅长的编程类型之一 |
| **3人团队可行性** | ★★★☆☆ — 卡牌平衡性调试和美术资源量大，3人时间紧凑但可行 |
| **独特性** | ★★★★☆ — 生旦净末丑做卡牌阵营，概念新鲜 |

**玩法：**
- 生旦净末丑五大职业作为卡牌阵营，玩家组建戏曲卡组进行对战
- 卡牌技能源自戏曲元素 — "变脸"可改变对手牌型、"甩水袖"可封印对方卡牌、"唱腔"可增益己方
- 收集角色卡牌会解锁该戏曲角色的真实背景故事（关羽、穆桂英、包拯等）
- **亮点：** 用年轻人熟悉的卡牌对战形式包装戏曲文化
- **开发要点：** 卡牌对战模板成熟，核心在卡牌设计和文化内容的深度结合

## Idea 7：《方言密室》— 方言文化

| 属性 | 评估 |
|------|------|
| **类型** | 密室逃脱 + 语音/文字解谜 |
| **难度** | ★★★★☆（中高） |
| **CodeBuddy 适配度** | ★★★☆☆ — 密室交互逻辑 AI 可写，但语音识别集成（方言）和音频素材是瓶颈 |
| **3人团队可行性** | ★★☆☆☆ — 方言语音识别实现难度大；退化为纯文字版则可行，但会削弱亮点 |
| **独特性** | ★★★★★ — 方言+密室逃脱，极具创意的组合 |

**玩法（推荐文字版）：**
- 你被困在一个由方言守护的古老宅院中，每道门锁都需要用当地方言的正确理解才能打开
- 玩家听到一句方言音频（或看到方言文字），通过选择正确的含义来解锁密室
- 每个房间对应一种方言（闽南语、粤语、吴语、四川话…）
- 解开一个方言房间后，学习该方言的文化冷知识
- **亮点：** 方言 + 密室逃脱，前所未见的组合
- **开发要点：** 建议做纯文字选择题版本（降低语音识别难度），方言音频预录即可
