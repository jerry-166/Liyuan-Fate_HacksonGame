# 赛题一：小红花游戏 — 公益主题 Ideas

> **背景：** 腾讯黑客松，CodeBuddy Agent 辅助，3人团队
> **引擎：** Godot / Unity

---

## Idea 1：《心跳快递》— 急救主题

| 属性 | 评估 |
|------|------|
| **类型** | 横版跑酷 + 点击决策 |
| **难度** | ★★☆☆☆（低） |
| **CodeBuddy 适配度** | ★★★★☆ — 跑酷物理和碰撞逻辑 AI 能批量产出 |
| **3人团队可行性** | ★★★★★ — 2D 横版制作周期短，1美术+2编程绰绰有余 |
| **独特性** | ★★★★☆ — 急救知识+跑酷的混搭少见，但需防"选择题感"太重 |

**玩法：**
- 你是一名外卖/快递员，送餐途中遇到路人突发状况（中暑、窒息、心脏骤停）
- 跑酷过程中出现"突发事件"→ 画面暂停 → 弹出多选决策（如"该做 CPR 还是打 120？"或"用海姆立克法还是等待救援？"）
- 选对救人成功，获得"小红花"积分；选错扣血，NPC 路人倒下会带来负面情绪反馈
- **正能量点：** 玩家潜移默化学会真实急救知识
- **开发要点：** 2D 横版 + 选择题交互，不需复杂物理引擎

## Idea 2：《熊猫星球》— 生物多样性

| 属性 | 评估 |
|------|------|
| **类型** | 模拟经营 + 拼图收集 |
| **难度** | ★★★☆☆（中） |
| **CodeBuddy 适配度** | ★★★★☆ — 资源管理数值系统和随机事件逻辑 AI 擅长构建 |
| **3人团队可行性** | ★★★★☆ — 文字/数据驱动为主，美术需求低 |
| **独特性** | ★★★☆☆ — 经营模拟品类成熟，差异在题材和科普内容 |

**玩法：**
- 你接管一个濒危物种保护区，通过合理分配资源（水、食物、医疗）维持生态平衡
- 每天出现随机事件（干旱、偷猎、物种入侵），玩家用有限的"爱心点数"调配资源
- 成功保护一个物种会解锁该物种的真实科普卡片（图鉴收集）
- **正能量点：** 了解濒危动物和生态保护知识
- **开发要点：** 文字/卡牌经营为主，辅以简单 2D 画面

## Idea 3：《银龄小镇》— 适老化关怀

| 属性 | 评估 |
|------|------|
| **类型** | 叙事养成 + 小游戏合集 |
| **难度** | ★★☆☆☆（低） |
| **CodeBuddy 适配度** | ★★★★★ — AVG 叙事引擎脚本和小游戏逻辑 AI 都很擅长 |
| **3人团队可行性** | ★★★★★ — 叙事驱动+简单 mini-game，AI 可批量产出对话分支 |
| **独特性** | ★★★★★ — 适老化题材在游戏领域非常稀缺，评委记忆点强 |

**玩法：**
- 你扮演社区志愿者，定期拜访几位独居老人，帮助他们解决生活难题
- 每个老人有"心情值"和"健康值"，通过陪聊（对话选择）、帮买药（记忆小游戏）、教用手机（拖拽教学）等小游戏提升数值
- 随着好感度提升，解锁老人的回忆故事（温情叙事）
- **正能量点：** 传递尊老爱老理念，了解老年生活困境
- **开发要点：** AVG 叙事 + 简单 mini-game，适合用 Ren'Py 或 Godot 快速实现

## Idea 4：《归途》— 阿尔茨海默症关怀

| 属性 | 评估 |
|------|------|
| **类型** | 解谜步行模拟器（2D俯视角 / 低多边形3D） |
| **难度** | ★★★★☆（中高） |
| **CodeBuddy 适配度** | ★★★★★ — 场景交互逻辑、记忆碎片系统、视觉滤镜效果 AI 都能高效生成 |
| **3人团队可行性** | ★★★☆☆ — 如果做 3D 版本需要美术功底；退化为 2D 俯视角则完全可行 |
| **独特性** | ★★★★★ — 用"体验疾病视角"做游戏的方式非常罕见，有强烈冲击力 |

**玩法：**
- 你以一个轻度阿尔茨海默症患者的视角，在家附近"迷路"，需要通过碎片化的记忆线索找到回家的路
- 世界是模糊、碎片化的 — 路牌文字会扭曲、熟悉的面孔认不出。玩家需要寻找"记忆锚点"（老歌、旧照片、特定的气味图案）来触发正确的路径
- 随着关卡推进，患者的症状加重，世界变得更加难以辨认
- **正能量点：** 让年轻人切身感受阿尔茨海默症患者的真实体验，唤起共情和关怀
- **开发要点：** 强烈推荐 2D 俯视角（降低美术门槛），用视觉滤镜和文字扭曲实现"认知障碍"效果

## Idea 5：《光能补给站》— 碳中和/环保

| 属性 | 评估 |
|------|------|
| **类型** | 三消 + 基地建设 |
| **难度** | ★★★☆☆（中） |
| **CodeBuddy 适配度** | ★★★★☆ — 三消匹配算法和消除判定是经典编程题，AI 可以秒出 |
| **3人团队可行性** | ★★★★☆ — 三消逻辑成熟，1人做数值+2人做经营部分 |
| **独特性** | ★★★☆☆ — 三消品类成熟，差异化在于环保主题包装 |

**玩法：**
- 你在一个被污染的未来世界运营一座"光能净化站"，通过消除污染来恢复生态
- 三消玩法中，消除的不是宝石而是"污染物"（塑料、废气、废水），每消除一个积攒"清洁能量"
- 清洁能量用来解锁和升级净化设施（风力发电机、污水处理厂、植树机器人）
- **正能量点：** 让玩家理解碳中和的运作逻辑 — 每一种排放都需要对应的"净化"去抵消
- **开发要点：** 三消逻辑非常成熟，换皮即可，核心精力放在经营系统和视觉风格

## Idea 6：《手语教室》— 无障碍沟通

| 属性 | 评估 |
|------|------|
| **类型** | 教学叙事游戏 / 点击选择 |
| **难度** | ★★☆☆☆（低） |
| **CodeBuddy 适配度** | ★★★★☆ — 选择题交互和叙事分支 AI 都能快速搭建 |
| **3人团队可行性** | ★★★★★ — 纯 2D 点选交互，几乎不需要复杂系统 |
| **独特性** | ★★★★☆ — 手语教学题材少见，但需注意玩法不要太像"网课" |

**玩法：**
- 玩家扮演一个聋哑学校的新生，从零学习手语，最终在校园活动中用手语交到朋友
- 屏幕上显示一个手语动作，玩家需要在限定时间内从 4 个选项中选择对应的含义
- 剧情模式：每天解锁新手语词汇，穿插校园叙事 — 被误解、交朋友、参加手语歌唱比赛
- **正能量点：** 推广手语和无障碍沟通理念
- **开发要点：** 选择题教学为主，可用选择分支叙事包装，手语动画可用逐帧序列帧

## Idea 7：《盲盒救援》— 流浪动物救助

| 属性 | 评估 |
|------|------|
| **类型** | 盲盒收集 + 放置养成 |
| **难度** | ★★☆☆☆（低） |
| **CodeBuddy 适配度** | ★★★★★ — 卡牌收集系统、放置养成数值 AI 非常擅长批量生产 |
| **3人团队可行性** | ★★★★★ — 轻度玩法，1人即可完成开发框架，2人做内容填充 |
| **独特性** | ★★★★☆ — "盲盒"包装正能量非常贴合年轻玩家，病毒传播潜力大 |

**玩法：**
- "盲盒"打开不是玩具，而是真实的流浪动物救助故事
- 玩家消耗虚拟币"救助点数"开启盲盒，每开一个解锁一只流浪动物从被发现→救助→康复→被领养的真实故事线
- 游戏内有收容所经营要素：建造狗舍、购买粮食、安排领养日
- 每只动物被领养后，进入玩家的"幸福图鉴"
- **正能量点：** 宣传"领养代替购买"理念
- **开发要点：** 卡牌收集 + 放置经营，资源管理为主，AI 批量生成动物故事卡片
