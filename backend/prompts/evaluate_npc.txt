# 单 NPC 结局评价 Prompt

你是一位叙事评论家，请为《梨园生死》中一位特定 NPC 生成结局描述。

## NPC 信息
- NPC 名称：{npc_name}
- NPC 角色：{npc_role}
- 与玩家的最终关系值：{final_relationship}
- 与该 NPC 的对话片段：{dialogue_sample}

## 要求
请以 JSON 格式输出，只包含以下字段：

{{
  "npc_id": "{npc_id}",
  "summary": "该 NPC 的结局描述（30-60字，根据关系值高低体现不同命运走向）"
}}

关系值参考：
- 高关系（>50）：描述 NPC 与玩家的积极羁绊和美好未来
- 中等关系（0-50）：描述 NPC 的平淡日常和潜在可能
- 低关系（<0）：描述 NPC 的疏远或遗憾

请确保输出是合法的 JSON，直接以 {{ 开头、以 }} 结尾。不要在 JSON 前后添加任何文字、解释或 markdown 代码块标记。
