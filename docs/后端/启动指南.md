# 后端启动指南

---

## 前置条件

- Python >= 3.10
- 推荐使用 [uv](https://docs.astral.sh/uv/)（超快速包管理器）

## 安装 uv（可选但推荐）

```powershell
pipx install uv
# 或
powershell -c "irm https://astral.sh/uv/install.ps1 | iex"
```

## 启动方式

### 1. 创建虚拟环境 & 安装依赖

```powershell
cd D:\workspace\A_Hackson_game\backend

# uv
uv venv
uv pip install -r requirements.txt

# 或用 pip
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

### 2. 配置环境变量（可选）

> 📎 **详见 [环境配置.md](./环境配置.md)** — 完整的环境变量列表和 API Key 管理。

```powershell
# TokenHub 接口地址
$env:LLM_BASE_URL = "https://dashscope.aliyuncs.com/compatible-mode/v1"
# 全局 fallback API Key
$env:TENCENT_LLM_API_KEY = "sk-你的key"
# 默认模型名
$env:LLM_MODEL = "deepseek-v4-flash"
```

### 3. 启动服务

```powershell
# uv
uv run python main.py

# 或普通 Python
python main.py
```

服务启动在 **http://localhost:8000**。

## 接口说明

| 方法 | 路径 | 说明 |
|------|------|------|
| `GET` | `/api/health` | 健康检查 |
| `POST` | `/api/game/start` | 创建游戏会话 |
| `GET` | `/api/game/{session_id}` | 获取游戏状态 |
| `POST` | `/api/dialogue` | NPC 对话（SSE 流式） |
| `POST` | `/api/game/{session_id}/evaluate` | 结局评价 |

Swagger 文档：**http://localhost:8000/docs**

## 常见问题

**Q: 调用返回 401 Unauthorized？**
A: 检查两项：
1. API Key 是否正确
2. `LLM_BASE_URL` 是否与 Key 对应的服务匹配（TokenHub vs 知识引擎原子能力）

**Q: 如何在 Swagger UI 中传入 API Key？**
A: 调用 `POST /api/game/start`，在请求体中填写：
```json
{
  "player_name": "玩家",
  "api_key": "sk-你的key",
  "model": "deepseek-v4-flash"
}
```
