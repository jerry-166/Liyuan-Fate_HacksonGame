# 《梨园生死》环境配置

> 管理所有环境变量和 API Key 配置，唯一的真相源。

---

## 环境变量一览

```bash
# .env (不提交到 Git)
TENCENT_LLM_API_KEY=sk-xxxxx          # 默认 LLM Key（可选，用户可覆盖）
TENCENT_LLM_BASE_URL=https://api.hunyuan.cloud.tencent.com/v1
LLM_MODEL=hunyuan-lite                 # MVP 用小模型省成本
LLM_MAX_CONCURRENCY=5                  # LLM 并发调用上限
SESSION_TTL_SECONDS=7200               # Session 内存 TTL（默认 2 小时）
DATABASE_PATH=./data/game.db
```

> **注意**: TokenHub 用户请设置 `LLM_BASE_URL=https://tokenhub.tencentmaas.com/v1`；旧版知识引擎用户请设置 `LLM_BASE_URL=https://api.lkeap.cloud.tencent.com/v1`。

## API Key 管理

### 优先级链

```
用户前端输入 → 环境变量 TENCENT_LLM_API_KEY → 错误提示
   (最高)           (fallback)               (无Key)
```

### 实现

```python
# config.py（由路由层在开始游戏时调用）
TENCENT_LLM_API_KEY = os.getenv("TENCENT_LLM_API_KEY")
# 当用户未在前端输入 Key 时使用此 fallback
```

**设计原则**：零存储（不写入数据库/日志/文件）、Session 隔离（每个游戏会话独立 Key）。

### 模型名称优先级

1. **session 级**：`game/start` 传入 `model`（如 `deepseek-v4-flash`）
2. **环境变量**：`$env:LLM_MODEL`
3. **默认值**：`deepseek-v4-flash`

### 接口地址优先级

1. **环境变量**：`$env:LLM_BASE_URL`
2. **默认值**：`https://tokenhub.tencentmaas.com/v1`
