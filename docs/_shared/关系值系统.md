# 关系值系统（单一真相源）

> 所有文档引用本文件获取关系值范围表、Clamp 机制等信息。

## 范围表

| 范围 | 含义 | NPC 行为表现 |
|------|------|-------------|
| -100 ~ -30 | 敌意 | 拒绝对话、言语攻击 |
| -30 ~ 0 | 冷淡 | 爱答不理、话中带刺 |
| 0 ~ 30 | 中立 | 正常交流 |
| 30 ~ 70 | 友善 | 主动分享、语气温和 |
| 70 ~ 100 | 信任 | 掏心窝子、透露秘密 |

## 关系值配置常量（来自 `config.py`）

```python
# 关系值总范围
RELATIONSHIP_CLAMP = (-100, 100)
# 单轮对话关系值变化边界
RELATIONSHIP_DELTA_CLAMP = (-5, 10)
# LLM 解析失败时的兜底增量
RELATIONSHIP_DEFAULT_DELTA = 3
# 触发关键事件的额外加分
RELATIONSHIP_EVENT_BONUS = 5
```

## 关系值 Clamp 机制

**策略：LLM 建议 + 规则兜底 + Clamp 限制**

```
每轮对话结束时：

1. LLM 返回 relationship_delta（建议值）
   ↓
2. Clamp 限制：将 delta 限制到 [RELATIONSHIP_DELTA_CLAMP] 范围
   - 正向上限 +10：防止一轮对话关系暴涨
   - 负向下限 -5：防止一轮对话关系暴跌
   ↓
3. 规则兜底增量（LLM 解析失败时）：
   - 每轮对话固定 RELATIONSHIP_DEFAULT_DELTA（+3）
   - 触发关键事件额外 +RELATIONSHIP_EVENT_BONUS（+5）
   ↓
4. 累加到 NPC.relationship，最终 clamp 到 RELATIONSHIP_CLAMP
```

### 事件关系值奖励

```python
EVENT_RELATIONSHIP_BONUS = {
    "chen_sent_to_xiaohua": 5,
    "xiaohua_mentioned_chen": 5,
    "chen_told_past": 10,
}
```

### 实现

```python
def apply_relationship_delta(npc, llm_delta, events):
    min_delta, max_delta = RELATIONSHIP_DELTA_CLAMP
    if llm_delta is not None:
        delta = max(min_delta, min(max_delta, llm_delta))
    else:
        delta = RELATIONSHIP_DEFAULT_DELTA
    for event_id in events:
        delta += EVENT_RELATIONSHIP_BONUS.get(event_id, 0)
    old = npc.relationship
    npc.relationship = max(-100, min(100, npc.relationship + delta))
    return npc.relationship - old
```
