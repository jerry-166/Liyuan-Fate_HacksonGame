class Stage(Enum):
    DISDAIN = 1    # 不屑 — 戏班众人冷眼相看
    UNDERSTAND = 2 # 了解 — 有人愿意说真心话
    CHOICE = 3     # 抉择 — 决定戏班的命运

## 阶段参数表

| 阶段 | 名称 | 色调 | BGM | 对话基调 | 切换条件 |
|------|------|------|-----|---------|---------|
| 1 | 不屑 | `cold` | melancholy | 冷漠疏离、话中带刺 | 游戏开始 |
| 2 | 了解 | `warm` | hopeful | 温和敞开、偶有真情 | 关系达标 OR 关键事件 OR LLM 判定 |
| 3 | 抉择 | `dramatic` | intense | 情感浓烈、命运抉择 | 关系达标 OR 关键事件 OR LLM 判定 |

## 阶段切换架构

### 加权判定引擎

```
每轮对话完成后（SSE done 事件生成时）：

   ┌──────────────────┐
   │  规则引擎 O(1)    │
   │                  │
   │ 条件1: any NPC   │
   │  relationship   │
   │  >= 50 → true   │
   │                  │
   │ 条件2: 关键事件   │
   │  已触发 → true   │
   └────────┬─────────┘
            │
     ┌──────┴───────┐
     │ 规则命中？     │
     └──┬──────┬────┘
        │      │
     Yes│      │No
        ▼      ▼
   立即切换  ┌──────────────────┐
            │ 独立 LLM 阶段判定  │
            │                  │
            │ Prompt（轻量）：   │
            │ "当前阶段{stage}， │
            │  关系值{rel}，    │
            │  已触发事件{evts}，│
            │  最近对话摘要。    │
            │  是否可推进？"     │
            │                  │
            │ 输出：            │
            │ {should_advance: │
            │  bool, reason}   │
            └────────┬─────────┘
                     │
              ┌──────┴───────┐
              │ LLM 判定推进？ │
              └──┬──────┬────┘
                 │      │
              Yes│      │No
                 ▼      ▼
        ┌────────────┐  跳过
        │ 连续判定计数 │
        │             │
        │ 连续 2 轮   │──── No ──→ 不切换，计数清零
        │ 都说推进？   │
        └──────┬──────┘
               │ Yes
               ▼
          执行阶段切换
```

| 判定来源 | 权重 | 切换条件 |
|---------|------|---------|
| 规则引擎 | **立即生效** | 任一条件达标 → 直接切换，无需 LLM 确认 |
| LLM 判定 | **需连续确认** | 连续 2 轮 LLM 独立判定均为"可推进" → 才执行切换 |
| 两者同时 | 规则优先 | 规则命中时不等 LLM 结果 |

### 切换伪代码

```python
async def advance_stage(session: GameSession, new_stage: Stage, reason: str):
    session.current_stage = new_stage
    for npc in session.npcs.values():
        npc.current_greeting = await generate_greeting(npc, new_stage)
    session.events_triggered.add(f"stage_{new_stage.value}_reached")
    session.stage_llm_consecutive = 0
    await db.update_session(session)

async def check_stage_advancement(session, events_this_turn):
    # 规则引擎
    if any(npc.relationship >= 50 for npc in session.npcs.values()):
        return await advance_stage(session, next_stage(...), "关系值达标")
    if any(e in KEY_STAGE_EVENTS for e in session.events_triggered):
        return await advance_stage(session, next_stage(...), "关键事件触发")
    # LLM 判定
    result = await llm_client.stage_check(...)
    if result.should_advance:
        session.stage_llm_consecutive += 1
        if session.stage_llm_consecutive >= 2:
            return await advance_stage(session, next_stage(...), result.reason)
    else:
        session.stage_llm_consecutive = 0
    return False
```
