# 核心数据模型（单一真相源）

> 所有文档引用本文件获取 GameSession、NPCState 等数据结构定义。

## GameSession

```python
from dataclasses import dataclass, field
from enum import Enum

class Stage(Enum):
    DISDAIN = 1    # 不屑
    UNDERSTAND = 2 # 了解
    CHOICE = 3     # 抉择

@dataclass
class GameSession:
    session_id: str
    player_name: str = "玩家"
    api_key: str = ""                    # 用户提供，仅存内存
    current_stage: Stage = Stage.DISDAIN
    npcs: dict[str, "NPCState"] = field(default_factory=dict)
    events_triggered: set[str] = field(default_factory=set)
    game_ended: bool = False
    ending_type: str | None = None
    ending_data: dict | None = None      # 缓存的结局评价
    stage_llm_consecutive: int = 0       # LLM 连续判定推进的轮数（≥2 才切换）
    last_active_at: float = field(default_factory=time.time)
    created_at: float = field(default_factory=time.time)
```

## NPCState

```python
@dataclass
class NPCState:
    id: str                    # "npc_chen"
    name: str                  # "陈师傅"
    role: str                  # "老琴师"
    scene: str                 # "tavern"
    position: tuple[int, int]  # (1200, 800)
    sprite_key: str            # "npc_chen_idle"
    relationship: int = 0      # -100 ~ 100
    is_available: bool = True
    current_greeting: str = ""
    dialogue_history: list[dict] = field(default_factory=list)
    last_options: list[str] = field(default_factory=list)
    dialogue_round_count: int = 0
```
